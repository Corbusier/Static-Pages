# README

## 简介
网站静态布局，不包含可交互内容。熟悉语义化标签，了解浏览器的兼容性问题，并且尝试解决。

其中：
1. 国美(包含index、list、detail)页面，处理了常见的兼容性问题，能够兼容到IE6浏览器。
2. WeHtml页面采用响应式布局，内容列表部分使用了弹性盒模型。
